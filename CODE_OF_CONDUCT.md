# Code of Conduct

## Our standards

Please help keep this project positive, healthy, and respectful.

Expected behavior:
- be respectful
- be kind
- communicate clearly
- keep discussion friendly

Unacceptable behavior:
- harassment
- personal attacks
- insulting or disrespectful language
- disruptive behavior

## Enforcement

Project maintainers are responsible for reviewing reported behavior and taking appropriate action to keep the community respectful.
