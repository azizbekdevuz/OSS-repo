# OSS-repo
22013143 Azizbek

A small Python practice repository created for an Open Source Software course assignment.

## Current status

This repository is currently a minimal educational exercise, not a production-ready open source project.

The files are useful only as a basic example of:
- creating a Git repository
- adding and editing files
- committing changes
- working with branches
- pushing code to GitHub

## Purpose

The main purpose of this repository is to practice basic open source and version control workflow concepts in a classroom setting.

## Repository structure

```text
OSS-repo/ (main branch)
├── .gitignore
├── README.md
├── main.py
├── test.py
└── test_feature.py
```

## Requirements

- Python 

No external libraries are required.

## How to run

Run the files directly with Python.

### Run the main script

```bash
python main.py
```

Expected output:

```text
Hello world
```

### Run the additional example files

```bash
python test.py
python test_feature.py
```
