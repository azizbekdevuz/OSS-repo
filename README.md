# OSS-repo

A small Python practice repository created for an Open Source Software course assignment.

## About

This repository is a minimal educational exercise for practicing basic Git and GitHub workflow in a classroom setting.

## Project files

- `main.py` — comprises of the source code for assignment 5
- `test.py` — prints a simple message
- `test_feature.py` — prints a simple message related to feature branch practice

## Purpose

The purpose of this repository is to practice:
- creating a repository
- adding and editing files
- committing changes
- working with branches
- pushing code to GitHub

## Requirements

- Python 3.x
- [Official docs] (https://docs.python.org/)

## How to run

```bash
python main.py
python test.py
python test_feature.py
```
